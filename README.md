## What This Is

A conversion-focused landing page and audit system for AI4Tradies. Tradies fill in the free audit form, receive a personalised report showing where their business is leaking money, and get a clear path to fixing it.

## Audit Reports

- `audits/painter_audit.md` — Profit leak audit for painters
- `audits/electrician_audit.md` — Profit leak audit for electricians
- `audits/hairdresser_audit.md` — Profit leak audit for hairdressers

## Template

`templates/audit_template.md` — Base template used to generate personalised audit PDFs.
